I have been trying to find a solution for this for a while but it hasn't been working. My friend Nathan has been working hard on the server and helping me out a lot.

So what is the problem?

If you don't know much about minecraft servers, you probably think that our bungeecord network is running off of one single process. That is not what how a bungeecord network operates. A network is made up a host proxy (running bungeecord) and several smaller spigot servers. The proxy just handles players and them connecting, switching servers, and disconnecting. The spigot servers actually handle the games and worlds. I have a single computer and that will be running a minecraft server and also will be used as a NAS. Forget about the NAS for now since I don't have any hard drives yet. Since a bungeecord network has to run on multiple processes, it would be hard to run a network easily on a non-graphical OS.
