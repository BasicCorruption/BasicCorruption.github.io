Bitrate Calculator

Steps to calculate

  • 1. Multiply video width by height to get pixel count (example: 2560 x 1080 = 604800)
  • 2. Multiply the pixel count by frame rate to get pixels per second (example: 604800 x 60 = 36288000)
  • 3. Multiply the pixels per second by 8 to get bitrate (example: 36288000 x 8 = 290304000)

Remember that recording software such as OpenBroadcasterSoftware (OBS) do not take in a bit per second bitrate. If you record using OBS, you must convert the format. Since it takes in a number for kilobits per second, you must divide the output number by 1000. So if I was using this calculator, my final number would be 290304 Kb/s (209 Mb/s). Please note that these numbers are not optimal for streaming. It will use up a lot network bandwidth and your stream might crash. Please do not use this calculator for streaming.
